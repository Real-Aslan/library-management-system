# Интернет дүкен (Java Console)

## Жоба туралы
Бұл жоба Java тілінде жасалған консольдік интернет дүкен болып табылады. Жүйе арқылы қолданушы тауарларды көріп, себетке қосып, тапсырыс бере алады.

## Мақсаты
Java бағдарламалау тілінде объектіге бағытталған тәсілді қолданып, қарапайым интернет дүкен жүйесін құру.

## Қолданылған технологиялар
- Java  
- OOP (Object-Oriented Programming)  
- ArrayList  
- Scanner  

## Мүмкіндіктері
- Тауарлар тізімін көру  
- Себетке тауар қосу  
- Себетті көру  
- Себеттен тауар өшіру  
- Себетті тазалау  
- Тапсырыс рәсімдеу  

## Жобаның құрылымы
- Item.java — тауар  
- BasketProduct.java — себет элементі  
- Buyer.java — сатып алушы  
- Purchase.java — тапсырыс  
- InternetDuken.java — негізгі файл  

## Іске қосу
```bash
javac *.java
java InternetDuken
