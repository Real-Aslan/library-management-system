Кітапхана басқару жүйесі (Java Console)
  Жоба туралы

Бұл жоба Java тілінде жасалған консольдік кітапхана басқару жүйесі болып табылады.
Жүйе арқылы қолданушы кітаптарды көре алады, жаңа кітап қоса алады, кітапты ала алады және қайтара алады.

  Мақсаты

Java бағдарламалау тілінде объектіге бағытталған тәсілді қолданып, қарапайым кітапхана басқару жүйесін құру және негізгі CRUD операцияларын үйрену.
